# WorldBook-Dynamic-Manager

你有过这样的想法吗？想让世界随着你的冒险一起成长，却总是卡在技术门槛上，折腾半天也搞不出想要的效果。

有没有想过，有一天，你手头那张角色卡，能真的“活”过来——不再是静态的设定集，而是一本会自己更新的动态世界书？

我曾经不止一次动过这个念头。搜了一圈发现，别人做的那些，都只能绑在他们自己的角色卡上用。而那种谁都能用、随便拿一张卡就能让它活起来的——我一个都没找到。

所以就有了这个。

不管你是写卡的作者，还是玩卡的玩家，这个脚本就是为你准备的。它只做一件事——让世界书自动更新。

用AI驱动，你可以自己设置什么时候触发更新。你看到什么、经历什么，世界书就会跟着增加什么。

路边发现一株血灵草？AI会帮你记进世界书，下次再遇到它，这个世界还记得你曾经来过。

脑补出一个新势力？AI也会帮你写进去，让这个设定成为世界的一部分。

世界不再是开篇写完就固定死的背景板，而是真正跟着你的脚步，一点点变得丰满起来。你每往前走一步，它就在你身后多铺开一寸。

说白了——你在完善你的世界，而它，也在回应你。

正式介绍·核心功能：

**一.智能目标定位**

通过 “目标世界书类型” 来控制AI的修改要写入哪本书。主要有四种模式：

1. 角色主世界书 (charPrimary)
   · 逻辑：自动找到当前角色正在使用的那本世界书（就是角色卡里绑定的那本）。
   · 适用场景：你想让AI为这个特定角色丰富其专属的世界设定，或单纯随着旅程更新内容
2. 角色附加世界书 (charAdditional)
   · 逻辑：除了角色主世界书，角色还可以关联多本附加世界书。你可以在这里指定一个具体的书名，让AI的修改都写入那本书。
   · 适用场景：你想为多个角色共享一个公共设定库。比如，你有一个叫“修仙界通用设定”的世界书，把它作为多个修仙角色的附加书，那么AI在跟任何修仙角色聊天时，对修仙界规则的补充都会汇总到这本公共书里。
3. 全局世界书 (global)
   · 逻辑：直接指定一个服务器/用户级别的全局世界书（就是在SillyTavern世界书管理界面里能看到的所有书）。
   · 适用场景：你想让AI维护一个跨越所有角色和聊天的宏大世界观。比如，一个“主神空间”设定，无论你和哪个角色卡聊天，AI发现的新规则都写入这本全局书。
4. 托管世界书 (managed)
   · 逻辑：这是一个非常聪明的“智能选择”模式。
     · 如果当前聊天已经关联了某本世界书（比如之前手动绑定的），它就使用那本。
     · 如果没有，它会自动创建一个新的世界书，名字通常是 角色名_WBM_日期。
   · 适用场景：你希望插件自动为你打理一切。完全不用操心世界书该放哪，它会自动创建、自动关联，可以作为聊天隔离的替代品（目前），你可以在不同聊天记录聊天时替换不同对应的关联世界书

**二.AI 自动管理世界书**

1. 自动审查与更新
   · 在对话进行到特定楼层（可自定义起始楼层和间隔）时，WBM 会自动将最近的聊天记录和当前世界书的全部内容打包，发送给一个专门配置的AI（如GPT-4）。
   · AI 会分析对话，判断哪些世界设定信息需要被记录、更新或删除。
2. AI 输出结构化指令
   · 如果AI认为需要更新世界书，它不会直接修改，而是按照插件规定的格式，输出一个包含具体指令的JSON代码块（被 <world_update> 标签包裹）。
   · 指令类型包括：
     · create: 创建一个全新的世界书条目。
     · update: 完全替换某个条目的内容。
     · delete: 删除一个条目。
     · patch: （最推荐） 对现有条目进行增量更新，例如追加内容、替换特定文本、修改关键词或调整条目属性（如深度、排序等）。这种方式非常节省Token。
3. 指令解析与执行
   · 插件会自动识别AI回复中的 <world_update> 指令，解析其中的JSON。
   · 然后，它会根据指令（patch、create等）自动修改SillyTavern中对应的世界书文件。

**三.核心工作模式与交互**

1. 两种工作模式
   · 内联模式 (Inline)：插件直接使用你正在对话的AI（主API）来执行审查任务。AI的回复中除了正常的对话内容，还会夹带<world_update>指令。插件执行指令后，会从AI消息中剥离这些指令，你在聊天界面看不到它们，体验很无缝。（我没测试过，这个功能无论在任何一张卡里都会消耗主api的注意力可能导致正文质量下降/我认为的）
   · 外部模式 (External)：你可以配置一个专门的API（如一个更便宜或更擅长总结的模型）在后台默默进行世界书的审查和更新，不影响你与主AI的对话。
